# ZAnimeList

This Project aims to take back control of your anime list which is probably on https://myanimelist.net/ or https://anilist.co/.


# Features
## Dashboard
The dashboard shows watched animes or animes that are plan to watch. They can be filtered based on the status and genres.


## Import
You can import or export directly from and to the following providers:
- https://myanimelist.net/
- https://anilist.co/
